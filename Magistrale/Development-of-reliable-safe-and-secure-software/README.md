# Development of Reliable, Safe and Secure Software

Materiale di studio per l'esame del corso **Development of Reliable, Safe and Secure Software** (cod. 1009069) — Laurea Magistrale in Scienze Informatiche, Università degli Studi di Parma.

**Docente**: Prof. Roberto Bagnara — 6 CFU, lingua inglese — [pagina ufficiale](https://corsi.unipr.it/it/ugov/degreecourse/189149).

**Esame**: progetto finale + colloquio orale (consegna elaborati almeno 15 giorni prima dell'appello).

## Struttura repo

- `claude_notes/` — appunti e materiale generato come supporto allo studio in attesa del materiale ufficiale dal docente. Da considerare provvisorio: una volta reperito il materiale del corso, sarà sostituito/integrato con un set di note basato su quello.
  - `info-corso.md` — scheda ufficiale del corso (obiettivi, programma, esame, bibliografia)
  - `email-bagnara.md` — bozza mail al docente per richiesta materiale
  - `appunti/` — note tematiche per argomento (11 capitoli)
  - `riassunti/` — cheatsheet di ripasso
